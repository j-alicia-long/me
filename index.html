Techstrav Volunteer Sorter


Description

I am currently working on a Python program that sorts over 300 volunteers into booths and optimal job assignments based on their preferred subject area and shift input data. This project is for Techstravaganza, an annual event organized by TWIST (led by me) that attracts hundreds of visitors each year. I got tired of sorting volunteers manually year after year, so I figured I’d use my coding skills to solve this problem!
The above image is an outreach flier that I created and designed.

  • Service: Software Development

Secret Santa Helper


Description

Secret Santa Helper was built for my high school Cheerleading team. Each week before a football game, our team would exchange small anonymous gifts with each other - like Secret Santa. We often had problems with drawing new names for the following week - often getting ourselves or the same person several times in a row and having to redraw. Thus, I used my coding skills to write a Java program that draws new names for each week - with no repeats - and sends the team their secret pal by email.
  • Date: September 2014
  • Service: Software Development

Evolve


Description

Inspired by my interest in AI and passion for Chemistry, Evolve is a mobile-friendly website that allows students to converse online in order to solve problems in Chemistry. If help is needed, the AI will provide a step-by-step solution of the problem. In the first few trials, feedback should be manually inputted by the users. This will allow the AI to learn and accurately provide solutions later on.
Evolve was built with Bootstrap, MongoDB, Sublime-Text, and JGrasp, and written in Python, HTML/CSS, and Javascript.
Through this project I learned how to create a database and integrate it with the website. This project was built for HackTJ 2.0.
  • Date: May 2015
  • Service: Web Development

EZMeet


Description

As a busy high school club leader, I often have trouble devising meetings that work with everyone’s schedule. This was the inspiration for EZMeet, a web application that provides the optimal locations and times for meetings based on info collected from users. Notifications are sent through email and text messaging.
Through this project I learned how to use Twilio, and gained deeper knowledge of PHP and HTML.
This project could potentially be implemented on Craigslist so people can find meeting points without disclosing their personal information to the other party.
This project was built for HackTJ 3.0.
  • Date: February 2016
  • Service: Web Development

BirdLife


Description

Bird Life is a Java parody of Duck Life, a popular internet game. In Bird Life, the user makes an account to collect, train, and compete their birds.
I’ve always been interested in game development and wanted to give this a go! Through this project I learned how to make user-friendly interfaces and implement Graphics in Java (also that Java is not the best for Graphics).
This project was built for my Foundations CS class.

Brave the Elements


Description

A simple web game inspired by my love for Chemistry, Brave the Elements is a periodic table memorization game. Players are asked to name all 118 elements in the periodic table within a time limit of 5 minutes. Through this project I learned how to make a Javascript timer, and how to better integrate Javascript/JQuery with HTML/CSS.
This project was built for my Web Development class. Check it out here on my TJ site!
  • Date: April 2014
  • Service: Web Development
